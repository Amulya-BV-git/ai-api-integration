AI API INTEGRATION: This project demonstrates how to integrate multiple AI APIs, such as Hugging Face, CoHere, and Groq, using Python. It takes user input and generates AI-based responses.
🚀Features: *Generate response using AI, * Supports multiple APIs , * Simple and Begineer-friendly
⚙ Requirements : *Python 3.11, *Libraries: requests, cohere, groq, API keys
📦 Installation: pip install -r requirement.txt, py -m pip install requests, py -m pip install cohere, py -m pip install groq
🔑 Setup(Windows): setx HUGGINGFACE_API_KEY"" , setx COHERE_API_KEY" " , setX GROQ_API_KEY""
▶ Usage: python huggingface_example.py, python cohere_example.py , python groq_example.py
